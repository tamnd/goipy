# v0.0.315 - 2026-04-28

`pickletools` module — fixture 315 for https://docs.python.org/3/library/pickletools.html. Full CPython 3.14 pickletools API surface.

**New `vm/stdlib_pickletools.go` — `buildPickletools()`:**

- **`opcodes`** — list of 68 `OpcodeInfo` instances covering all CPython pickle opcodes (INT through BINPERSID), each with `name`, `code`, `arg`, `stack_before`, `stack_after`, `proto`, `doc` attributes
- **`OpcodeInfo`** — class with 7 attributes; all 68 opcodes built as instances
- **`StackObject`** — class; 18 singleton instances (`int_or_bool`, `int`, `bytes`, `str`, `list`, `dict`, `tuple`, `set`, `frozenset`, `any`, `mark`, `stackslice`, etc.)
- **`ArgumentDescriptor`** — class with `name`, `n`, `reader`, `doc`; 24 singletons covering all argument types (`uint1`, `uint2`, `uint4`, `uint8`, `int4`, `bytes1-8`, `string1/4/nl`, `float8/nl`, `long1/4`, `unicodestring*`, `decimalnl_*`, `bytearray8`)
- **`_Example`** — stub class
- **`decode_long(data)`** → `int` — real little-endian two's complement implementation
- **`genops(pickle)`** → empty list stub (no pickle parser in goipy)
- **`dis(pickle, out=None, ...)`** → `None` stub
- **`optimize(p)`** → bytes stub (returns input bytes unchanged)
- **24 `read_*` functions** — all callable stubs (`read_uint1` through `read_unicodestringnl`)
- **`__all__`** set to `['dis', 'genops', 'optimize']`

Registered `"pickletools"` in `vm/asyncio.go`.
