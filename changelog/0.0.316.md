# v0.0.316 - 2026-04-28

Windows-specific stdlib modules, fixture 316. Covers all four modules
listed on https://docs.python.org/3/library/windows.html.

New files:

**vm/stdlib_msvcrt.go** -- `msvcrt` module with lock constants (LK_UNLCK
through LK_NBRLCK), SEM_* error mode flags, CRT_ASSEMBLY_VERSION, and
stubs for locking/setmode/open_osfhandle/get_osfhandle, the getch/getwch
family, putch/putwch/ungetch/ungetwch, heapmin, SetErrorMode/GetErrorMode.

**vm/stdlib_winreg.go** -- `winreg` module with all HKEY_* root constants,
KEY_* access flags, REG_* value type constants, an HKEYType class with
Close/Detach/handle, and all 22 registry functions as OSError stubs
(ConnectRegistry, CreateKey, OpenKey, QueryValue, etc.). error points to
OSError as the documented alias.

**vm/stdlib_winsound.go** -- `winsound` module with all SND_* and MB_*
constants including the three new CPython 3.14 additions (SND_SENTRY,
SND_SYNC, SND_SYSTEM, MB_ICON* aliases), plus Beep/PlaySound/MessageBeep
stubs.

**vm/stdlib_msilib.go** -- `msilib` module with all type_* bitmask
constants, AMD64/Win64 flags, stub classes (Table, CAB, Directory, Binary,
Feature, Control, RadioButtonGroup, Dialog, _Unspecified), and function
stubs for change_sequence/add_data/add_stream/init_database/add_tables;
make_id returns its input unchanged, gen_uuid returns a zero UUID string.

All four registered in vm/asyncio.go. Expected output crafted from known
implementation values since CPython cannot import these on non-Windows.
