# v0.1.0 — 2026-04-29

"Run any CPython 3.14 .pyc" — the v0.1.0 contract.

Closes the gap audit at `notes/Spec/1500/1530_goipy_v01_gap_audit.md`.
Tag is at the same commit as v0.0.346; v0.1.0 marks the audit
green, not new code.

## What v0.1.0 promises

A `.pyc` produced by CPython 3.14 (`python3.14 -m compileall -b`)
loads and runs under goipy with semantics matching CPython for
the audit corpus. The 346 fixtures in `internal/testdata/` exercise
the surface; CI compiles them on every push.

## Hard out-of-scope (explicit non-goals)

- `compile()` / `exec(str)` / `eval(str)` actual source compilation
  (needs a Python source parser; the gocopy project is building
  one — integration is a v0.1.x or v0.2.x task).
- `weakref` GC tracking (Go runtime has no native weak refs;
  `runtime.AddCleanup` makes it achievable post-v0.1.0).
- `__del__` finalisation timing (Go GC is non-deterministic; not
  fixable without a refcount-on-Go-GC re-architecture).
- C extensions / `PyObject *` / cgo.
