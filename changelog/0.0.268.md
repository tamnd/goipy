# v0.0.268 - 2026-04-27

`trace` ‚Äî implements the Python execution tracing and coverage module from https://docs.python.org/3/library/trace.html.

**`trace.Trace(count, trace, countfuncs, countcallers, ignoremods, ignoredirs, infile, outfile, timing)`:** all constructor parameters accepted and stored; `infile` is read at construction (JSON); `outfile` is written when `results()` is called.

**`Trace.runfunc(func, *args, **kwds)`:** calls `func(*args, **kwds)` directly via `callObject`; if `countfuncs=1`, records `(filename, module, funcname)` in the accumulated `calledfuncs` data; returns the function's return value; propagates exceptions unchanged.

**`Trace.run(cmd)`** / **`Trace.runctx(cmd, globals, locals)`:** no-op stubs (goipy has no `sys.settrace`).

**`Trace.results() ‚Üí CoverageResults`:** returns a `CoverageResults` instance populated with accumulated data; writes `outfile` if configured.

**`trace.CoverageResults`:**
- `counts` ‚Äî empty dict (no line-level tracing in goipy)
- `calledfuncs` ‚Äî dict of `(filename, module, funcname)` 3-tuples ‚Üí `1`, populated by `runfunc` when `countfuncs=1`
- `callers` ‚Äî empty dict
- `update(other)` ‚Äî merges `other.calledfuncs` into self (deduplicates keys)
- `write_results(show_missing, summary, coverdir, ignore_missing_files)` ‚Äî no-op stub
