# Changelog: v0.0.001 to v0.0.050

## v0.0.010 - 2026-04-20

Initial release. The eval loop covers the core CPython 3.14 instruction set.
Ten fixtures: arithmetic, comparisons, strings, lists, control flow, functions,
dicts, classes, exceptions, and built-ins. No stdlib yet.

## v0.0.025 - 2026-04-20

`with` statements, `super()`, descriptor protocol (`__get__`/`__set__`/
`__delete__`), f-string format specs, `match`/`case` with all pattern types,
synchronous generators, walrus operator, slicing, comprehensions, and
sequence unpacking. Also covers stress versions of all of the above.

## v0.0.026 - 2026-04-20

Descriptor stress tests. The `__class__` cell is now plumbed correctly so
`super()` resolves through deep inheritance chains.

## v0.0.028 - 2026-04-20

Pattern matching edge cases: guard conditions, nested patterns, `as`-patterns,
and deep destructuring (fixture 28).

## v0.0.030 - 2026-04-20

Format spec polish: width, fill, align, sign, `z`, `#`, `0`, precision, and
all type codes. Edge cases around zero-padding and alternate form.

## v0.0.032 - 2026-04-20

Generator stress tests and deep generator chains. `yield from` across nested
generators, `send()`, `throw()`, and `close()` with cleanup frames.

## v0.0.035 - 2026-04-20

`async def` / `await`, async generators, `async for`, and `async with`. A
minimal event loop drives the fixtures; no `asyncio` module yet.

## v0.0.038 - 2026-04-21

Filesystem imports: single-module `.pyc` loading, relative imports, and
transitive imports across multiple files.

## v0.0.040 - 2026-04-21

Package imports: `__init__.py`, sub-packages, and relative `from pkg import x`
across two levels.

## v0.0.042 - 2026-04-21

`importlib`: `import_module`, `reload`, and `find_spec`.

## v0.0.044 - 2026-04-21

`complex` type: arithmetic, `abs`, `conjugate`, polar/rectangular conversion.

## v0.0.046 - 2026-04-21

`frozenset` as a distinct hashable type: set operations, `issubset`,
`issuperset`, `__hash__`.

## v0.0.048 - 2026-04-21

`bytearray`: mutable byte sequences, slice assignment, `append`, `extend`,
`decode`, format-style operations.

## v0.0.050 - 2026-04-21

`memoryview` over `bytes` and `bytearray`: slicing, format, shape, strides,
`tobytes`, `tolist`, `cast`.
