# v0.0.320

Nine Unix-specific stdlib modules from https://docs.python.org/3/library/unix.html are
now stubbed out: posix, pwd, grp, termios, tty, pty, fcntl, resource, syslog.

- posix -- constants (CLD_*, EX_*, F_LOCK/OK/RWX, NGROUPS_MAX, O_* open flags, WNOHANG,
  SCHED_*, AT_*, RTLD_*, PRIO_*, and more) plus ~150 callable stubs for file ops,
  process control, signals, env, and stat helpers.
- pwd -- struct_passwd with the seven standard fields; getpwuid(0) returns a minimal root
  record; getpwnam always raises KeyError; getpwall returns empty list.
- grp -- struct_group with four fields; getgrgid/getgrnam always raise KeyError;
  getgrall returns empty list.
- termios -- 196 macOS/BSD constants; tcgetattr stub returning a 7-element list;
  tcsetattr/tcdrain/tcflush/tcflow/tcsendbreak/tcsetwinsize return None;
  tcgetwinsize returns (24, 80); error aliased to OSError.
- tty -- re-exports all termios constants; setraw/setcbreak return None.
- pty -- CHILD/STDIN_FILENO/STDOUT_FILENO/STDERR_FILENO; openpty/fork/spawn raise OSError.
- fcntl -- 31 constants (F_DUPFD, F_GETFD/FL, F_RDLCK/WRLCK, FD_CLOEXEC, LOCK_* etc.);
  fcntl/flock/ioctl/lockf stubs.
- resource -- RLIMIT_* (10 values), RLIM_INFINITY, RUSAGE_SELF/CHILDREN; struct_rusage
  with 16 ru_* fields; getrusage returns zeroed struct; getrlimit returns (0, RLIM_INFINITY);
  setrlimit returns None; getpagesize returns 4096.
- syslog -- ~36 LOG_* constants (levels, facilities, options); syslog/openlog/closelog/
  setlogmask stubs; LOG_MASK and LOG_UPTO implemented with real bit arithmetic.

Fixture 320 covers all nine modules with 185 assertions.
