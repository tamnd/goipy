# Changelog: v0.0.101 to v0.0.150

## v0.0.104 - 2026-04-22

`re` module: full flag set (`IGNORECASE`, `MULTILINE`, `DOTALL`, `VERBOSE`,
`ASCII`, `LOCALE`), `Pattern` methods, `Match` methods. `string.templatelib`:
Template/Interpolation/convert (PEP 750).

## v0.0.107 - 2026-04-22

`re` expanded: `Pattern.fullmatch`, `re.split`, `re.subn`, `re.escape`,
iterator protocol, named groups, lookaheads, lookbehinds, backreferences.

## v0.0.108 - 2026-04-22

`difflib` full coverage: `SequenceMatcher` with `get_matching_blocks`,
`get_opcodes`, `ratio`; `context_diff`; `restore`; `Differ`.

## v0.0.109 - 2026-04-22

`textwrap` full coverage: `TextWrapper` constructor options, `fill`, `wrap`,
`dedent`, `indent`, `shorten`, long-word handling, whitespace collapsing.

## v0.0.110 - 2026-04-22

`unicodedata`: `lookup`, `name`, `category`, `bidirectional`, `combining`,
`east_asian_width`, `mirrored`, `decomposition`, `normalize` (NFC/NFD/NFKC/
NFKD), `is_normalized`, `unidata_version`.

## v0.0.111 - 2026-04-22

`stringprep` (RFC 3454): all 31 tables (`in_table_b1` through `in_table_d2`),
`map_table_b2`, `map_table_b3`.

## v0.0.112 - 2026-04-22

`readline`: `get_line_buffer`, `insert_text`, `redisplay`, `parse_and_bind`,
`set_completer`, `get_completer`, `add_history`, `get_history_item`,
`get_current_history_length`.

## v0.0.113 - 2026-04-22

`rlcompleter`: `Completer` class with `complete` method, attribute and keyword
completion.

## v0.0.114 - 2026-04-22

`struct` full coverage: all format codes (`b`, `B`, `h`, `H`, `i`, `I`, `l`,
`L`, `q`, `Q`, `f`, `d`, `s`, `p`, `?`, `x`), byte-order prefixes, alignment,
`Struct` class.

## v0.0.115 - 2026-04-22

`codecs`: `encode`, `decode`, `lookup`, `open`, incremental encoder/decoder,
standard ASCII/UTF-8/UTF-16/Latin-1/CP437 codecs.

## v0.0.116 - 2026-04-22

`datetime`: `date`, `time`, `datetime`, `timedelta`, `timezone`, `tzinfo`.
Arithmetic, comparison, `strftime`, `strptime`, ISO format, `fromisoformat`.

## v0.0.117 - 2026-04-22

`zoneinfo`: `ZoneInfo`, `available_timezones`, `ZoneInfoNotFoundError`. Uses
Go's `time.LoadLocation` as the backend.

## v0.0.118 - 2026-04-22

`calendar`: full coverage including `TextCalendar`, `HTMLCalendar`,
`monthcalendar`, `setfirstweekday`, leap-year utilities.

## v0.0.119 - 2026-04-22

`collections` full stdlib coverage: `deque` rotate/maxlen, `OrderedDict`
move_to_end, `Counter` most_common/subtract, `ChainMap` maps/parents,
`namedtuple` _make/_asdict/_replace, `defaultdict` missing-key.

## v0.0.120 - 2026-04-22

`collections.abc`: `isinstance` checks for `Mapping`, `Sequence`, `Set`,
`Callable`, `Iterable`, `Iterator`, `Generator`, `Coroutine`, `Awaitable`,
`AsyncIterable`, `AsyncIterator`, `AsyncGenerator`.

## v0.0.121 - 2026-04-22

`heapq` full coverage: `heapify`, `heapreplace`, `heappushpop`, `merge`,
`nlargest`, `nsmallest` with key.

## v0.0.122 - 2026-04-22

`bisect` full coverage: `bisect_left`, `bisect_right`, `insort_left`,
`insort_right`, all with `key` argument.

## v0.0.123 - 2026-04-22

`array`: typed homogeneous sequences. All type codes (`b`, `B`, `u`, `h`,
`H`, `i`, `I`, `l`, `L`, `q`, `Q`, `f`, `d`), `frombytes`, `tobytes`,
`tofile`, `fromfile`, `fromlist`, `tolist`.

## v0.0.124 - 2026-04-22

`weakref`: `ref`, `proxy`, `WeakValueDictionary`, `WeakKeyDictionary`,
`WeakSet`, `finalize`. Callbacks fire when the target is collected.

## v0.0.125 - 2026-04-22

`types`: `SimpleNamespace`, `MappingProxyType`, `ModuleType`, `FunctionType`,
`LambdaType`, `CodeType`, `GeneratorType`, `CoroutineType`, `AsyncGeneratorType`,
`UnionType`.

## v0.0.126 - 2026-04-23

`copy`: `copy`, `deepcopy` with cycle detection and `__copy__`/`__deepcopy__`
protocol support. `__reduce__`, `__reduce_ex__`, `__getstate__`/`__setstate__`.

## v0.0.127 - 2026-04-23

`pprint`: `PrettyPrinter` with indent, width, depth, compact, sort_dicts;
`pformat`, `pp`, `isreadable`, `isrecursive`, `saferepr`.

## v0.0.128 - 2026-04-23

`reprlib`: `Repr` class with customizable limits; `aRepr` singleton;
`recursive_repr` decorator.

## v0.0.129 - 2026-04-23

`enum`: `Enum`, `IntEnum`, `Flag`, `IntFlag`, `StrEnum`, `auto`, `unique`,
`member`, `nonmember`. `_generate_next_value_`, iteration, containment.

## v0.0.130 - 2026-04-23

`graphlib`: `TopologicalSorter` with `prepare`, `get_ready`, `done`,
`is_active`, `static_order`. Cycle detection via `CycleError`.

## v0.0.131 - 2026-04-23

`numbers`: abstract numeric tower: `Complex`, `Real`, `Rational`, `Integral`.
`isinstance` checks against user-defined numeric types.

## v0.0.132 - 2026-04-23

`math` full coverage: `comb`, `perm`, `isqrt`, `ulp`, `nextafter`, `dist`,
`prod`, `lcm`, `gcd` (multi-arg), `cbrt`, `exp2`, `log2`, `log10`.

## v0.0.133 - 2026-04-23

`cmath`: `phase`, `polar`, `rect`, `exp`, `log`, `log10`, `sqrt`, `sin`,
`cos`, `tan`, `asin`, `acos`, `atan`, `sinh`, `cosh`, `tanh`, `asinh`,
`acosh`, `atanh`, `isfinite`, `isinf`, `isnan`, `isclose`.

## v0.0.134 - 2026-04-23

`localcontext`, `DefaultContext`, `BasicContext`, `ExtendedContext`.

## v0.0.135 - 2026-04-23

`fractions`: `Fraction` with exact rational arithmetic, `limit_denominator`,

## v0.0.136 - 2026-04-23
