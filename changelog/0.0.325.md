# v0.0.325

`termios` module gets a dedicated deep fixture (325) and four constant fixes.

The module was stubbed in fixture 320 with all 197 constants present but
four had wrong values, and seven pseudo-constants were exported that don't
exist in CPython's termios module.

Constant fixes:

- `CCTS_OFLOW`: 131072 -> 65536 (0x10000)
- `CRTS_IFLOW`: 65536 -> 131072 (0x20000)  (these two were swapped)
- `CRPRNT`: 32 -> 18 (0x12)
- `TABDLY`: 4 -> 3076 (0xC04, the full TAB bitmask)

Removed non-CPython constants: `IFLAG`, `OFLAG`, `CFLAG`, `LFLAG`,
`ISPEED`, `OSPEED`, `CC` -- these were never part of the real termios
module and should not be exported.

Fixture 325 asserts all 197 constants with exact macOS CPython values,
verifies the seven removed constants are absent, checks all 8 functions
are callable, and exercises tty operations under `try/except termios.error`.
