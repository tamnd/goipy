# Changelog: v0.0.051 to v0.0.100

## v0.0.052 - 2026-04-21

Long-tail builtins: `pow`, `format`, `ascii`, `slice`, `dir`, `delattr`,
`hasattr`, `getattr`, `setattr`, `callable`, `isinstance`, `issubclass`.

## v0.0.054 - 2026-04-21

`__dunder__` methods on user classes: `__repr__`, `__str__`, `__bool__`,
`__len__`, `__iter__`, `__next__`, comparison dunders, hash.

## v0.0.056 - 2026-04-21

Complete dunder surface: in-place operators (`__iadd__` etc.), reflected
operators (`__radd__` etc.), `__contains__`, `__index__`, `__enter__`,
`__exit__`.

## v0.0.058 - 2026-04-21

Descriptor protocol and class-creation hooks: `__init_subclass__`,
`__set_name__`, `__class_cell__`. Descriptors on both instances and classes.

## v0.0.060 - 2026-04-21

`functools`: `reduce`, `partial`, `lru_cache`, `cached_property`,
`total_ordering`, `wraps`, `singledispatch`. `itertools`: full coverage
including `chain`, `product`, `combinations`, `permutations`, `groupby`.

## v0.0.062 - 2026-04-21

`collections`: `deque`, `OrderedDict`, `Counter`, `ChainMap`, `namedtuple`,
`defaultdict`. `operator`: all comparison, math, and item-access operators.

## v0.0.064 - 2026-04-21

`math`: full function set including `comb`, `perm`, `isqrt`, `ulp`, `dist`.
`heapq`: `heappush`, `heappop`, `nlargest`, `nsmallest`. `bisect`. `random`.

## v0.0.066 - 2026-04-21

`json`: encoder, decoder, `loads`, `dumps`, indent/separators/sort_keys.
`re`: `compile`, `search`, `match`, `findall`, `sub`, flags. `string`:
Template. `copy`: `copy`, `deepcopy`.

## v0.0.068 - 2026-04-21

`io`: `StringIO`, `BytesIO`, `FileIO`. `hashlib`: MD5, SHA-1, SHA-256,
`sha3_*`, `blake2`. `base64`: encode/decode variants. `textwrap`: `fill`,
`wrap`, `dedent`, `indent`, `shorten`.

## v0.0.070 - 2026-04-21

`struct`: `pack`, `unpack`, `calcsize`, all format codes. `csv`: reader,
writer, DictReader, DictWriter, dialects. `urllib.parse`: `urlparse`,
`urlencode`, `quote`, `unquote`. `zlib`: compress/decompress, crc32, adler32.

## v0.0.072 - 2026-04-21

`binascii`: hexlify, unhexlify, b2a/a2b variants. `uuid`: UUID1/UUID4/UUID5.
`hmac`: new, digest, compare_digest. `secrets`: token_bytes, token_hex,
token_urlsafe, compare_digest.

## v0.0.074 - 2026-04-21

`difflib`: SequenceMatcher, `ndiff`, `unified_diff`, `context_diff`, HtmlDiff.
`shlex`: `split`, `quote`, `join`. `gzip`: open, GzipFile. `fnmatch`: filter,
translate.

## v0.0.076 - 2026-04-21

`statistics`: mean, median, mode, stdev, variance, pstdev, NormalDist.
`calendar`: month, monthcalendar, isleap, monthrange. `pprint`: PrettyPrinter,
indent, width, depth. `html`: unescape, entities.

## v0.0.078 - 2026-04-21

Line table decoder for CPython 3.14 bytecode. `traceback`: format_exc,
print_exc, extract_tb, StackSummary. `sys`: argv, path, modules, exc_info,
version, platform. `__await__` dispatch through coroutines.

## v0.0.080 - 2026-04-22

Missing built-ins added: `globals`, `locals`, `vars`, `object`, `input`,
`aiter`, `anext`, `open` (returning a real file handle), `os` module stub.

## v0.0.085 - 2026-04-22

Seven missing CPython 3.14 opcodes implemented. `__init__` return-value check.
`bytes.decode` encoding argument. `list.sort` `key` parameter. New fixtures:
`async for`, PEP 526 variable annotations, t-strings (PEP 750), `__init__`
checks, opcode stress tests.

## v0.0.086 - 2026-04-22

All built-in constants: `False`, `True`, `None`, `NotImplemented`,
`Ellipsis`, `__debug__`.

## v0.0.091 - 2026-04-22

Full `stdtypes` coverage from docs.python.org: all `str`, `int`, `float`,
`list`, `tuple`, `dict`, `set`, `bytes` methods per the Python 3.14 reference.

## v0.0.093 - 2026-04-22

Complete built-in exception hierarchy per docs.python.org/3/library/exceptions.
All exception classes instantiate correctly; `args`, `__cause__`, `__context__`,
`__suppress_context__` work as expected.

## v0.0.099 - 2026-04-22

Thread-safety fixtures: concurrent access to `list`, `dict`, `set`,
`bytearray`, and `memoryview` from multiple goroutines. The `threading` module
with `Lock`, `RLock`, `Condition`, `Semaphore`, `Event`, and `Barrier`, all
backed by Go sync primitives.

## v0.0.100 - 2026-04-22

`string` module: `Formatter` (full format spec mini-language), `Template`
(safe_substitute), string constants (`ascii_letters`, `digits`, etc.),
format spec alignment and fill character edge cases.

## v0.0.103 - 2026-04-22

`string.Formatter` extended tests. `string.Template` with custom delimiters.
`string.templatelib` (PEP 750): `Template`, `Interpolation`, `convert` for
`!r`/`!s`/`!a` conversions.
