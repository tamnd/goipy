# v0.0.317 - 2026-04-28

Dedicated fixture for the msvcrt module, fixture 317.
Full coverage of https://docs.python.org/3/library/msvcrt.html.

Fixture 316 already added the msvcrt stub as part of the bulk Windows
modules pass. This release adds a thorough standalone test that exercises
every documented symbol individually.

New `internal/testdata/317_msvcrt_deep.py` tests:

- All five LK_* locking constants (value and int type)
- All four SEM_* error mode constants (value and int type), including
  verifying SEM_FAILCRITICALERRORS|SEM_NOGPFAULTERRORBOX == 3
- CRT_ASSEMBLY_VERSION is a non-empty string
- locking(), setmode(), open_osfhandle(), get_osfhandle() callable and
  return the right types (None / int)
- kbhit(), getch(), getche(), getwch(), getwche() callable and return
  bool / bytes / str as documented
- putch(), putwch(), ungetch(), ungetwch() callable and return None
- heapmin() callable and returns None
- SetErrorMode() and GetErrorMode() callable and return int

Expected output crafted from known implementation values since CPython
cannot import msvcrt on non-Windows.
